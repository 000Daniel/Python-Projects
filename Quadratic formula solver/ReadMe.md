## ReadMe - Info
<br />
This script can accept three(3) numbers to solve a quadratic formula with.
<br />
<br />
### Usage:
   simply type in console: ```python3 Quadratic-Formula.py a b c```
   replace the 'a', 'b' and 'c' with your desired numbers.
